# Scala style (Scalaguide)

This project uses the **Scala style guide** for all Scala code.

- **Guide:** [Official Scala Style Guide](https://docs.scala-lang.org/style/)
- **Enforcement:** [Scalastyle](http://www.scalastyle.org/) via [scalastyle-config.xml](scalastyle-config.xml)
- **Details:** [docs/scala-style-guide.md](docs/scala-style-guide.md)

Run `gbx:lint:scalastyle` locally to check style before pushing.
