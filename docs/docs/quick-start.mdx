# Quick Start

This guide will help you get started with GeoBrix quickly after installation. All code below is copy-paste executable when you use the sample data paths shown.

:::tip Set up sample data first
The examples on this page use **sample data paths** (e.g. `/Volumes/.../geobrix-examples/nyc/...`). To run them as-is, set up sample data first: **[Sample Data Guide](./sample-data/overview)** — then use the same paths or volume in your notebook.
:::

## Prerequisites

- [Install GeoBrix](./installation) on your Databricks cluster.
- (Recommended) [Set up sample data](./sample-data/overview) so the snippets below run without changing paths.

:::note
You do not need to register functions if you are only using the included readers.
:::
