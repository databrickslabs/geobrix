### File Geodatabase - NYC Multi-Feature Class

:::info Complete bundle
Optional: Create with the script below. GeoBrix can read FileGDB (OpenFileGDB driver). The Complete setup script creates a GeoPackage instead; use this section if you need FileGDB format.
:::

Multi-feature class File Geodatabase (Esri format) created with **OpenFileGDB driver** (GDAL 3.6+).

:::tip OpenFileGDB Driver (GDAL 3.6+)
The **OpenFileGDB driver** provides full read/write support for FileGDB since GDAL 3.6 and is included in standard GDAL installations. No proprietary Esri SDK required!

**GeoPackage** is still recommended for new projects due to its open standard and wider tool support, but FileGDB creation now works out-of-the-box with modern GDAL.
:::

:::info OpenFileGDB Driver (GDAL 3.6+)
**Full read/write support** is available with the OpenFileGDB driver since GDAL 3.6:
- ✅ **Read**: Works with any GDAL version
- ✅ **Write**: Available since GDAL 3.6+ (included in standard distributions)
- ✅ **No proprietary SDK required** (unlike the old FileGDB driver)

**Source**: [GDAL OpenFileGDB Documentation](https://gdal.org/en/stable/drivers/vector/openfilegdb.html)

GeoPackage is still recommended for new projects due to its open standard, but FileGDB creation now works out-of-the-box with modern GDAL!
:::
